# pillagers

[Latest Demo](http://s3.amazonaws.com/superjoe/temp/pillagers/index.html)

![](http://i.imgur.com/MVwDAeq.png)

## Developing

 1. Install Node.js and libcairo-dev. Both are probably be in your
    operating system's package manager.
 2. `npm run dev`
